WeatherApp is a simple and user-friendly Python command-line application designed to fetch and display current weather information for any specified city.
How to Run:

Prerequisites: Ensure you have Python installed on your system. Additionally, if you are running the Dockerized version, Docker needs to be installed.

Running the Application Locally:

Download or clone the repository to your local machine.
Navigate to the application directory in your terminal or command prompt.
If running for the first time, install required packages: pip install -r requirements.txt.
Run the application: python assignment4.py.
When prompted, enter the name of the city you want to check the weather for.
